# Final Shop Frontend

Este es el proyecto frontend de la aplicación Final Shop, diseñado con una paleta de colores en negro y blanco, asegurando un diseño profesional y elegante.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

- **public/**: Contiene archivos estáticos, incluyendo `index.html`, que es la entrada principal de la aplicación.
- **src/**: Contiene el código fuente de la aplicación, incluyendo componentes, vistas, servicios y estilos.
- **package.json**: Configuración de npm, incluyendo dependencias y scripts.
- **tsconfig.json**: Configuración de TypeScript.
- **vite.config.ts**: Configuración de Vite para la construcción y el servicio de la aplicación.
- **.gitignore**: Archivos y carpetas que deben ser ignorados por Git.
- **README.md**: Documentación del proyecto.

## Instalación

Para instalar las dependencias del proyecto, ejecuta el siguiente comando en la raíz del proyecto:

```
npm install
```

## Uso

Para iniciar la aplicación en modo de desarrollo, utiliza el siguiente comando:

```
npm run dev
```

Esto iniciará un servidor de desarrollo y podrás acceder a la aplicación en `http://localhost:3000`.

## Construcción

Para construir la aplicación para producción, ejecuta:

```
npm run build
```

Esto generará los archivos estáticos en la carpeta `dist`.
